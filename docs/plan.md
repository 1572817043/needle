# Needle 第一版开发规划

## 目标

构建 macOS 优先的 AI 本地音乐助手：用户输入想听的歌，App 搜索 B 站公开视频，AI 筛选候选，用户试听或打开原视频确认后，用本机 `yt-dlp + ffmpeg` 转成 `m4a`，保存到 `~/Music/Needle` 并加入“我的歌曲”。

## 第一版范围

- 技术栈：Tauri v2 + React + TypeScript + Vite。
- 来源：只支持 B 站公开视频，不做登录、cookie、YouTube、插件源。
- AI：OpenAI 兼容接口，支持获取模型，失败时允许手动输入模型名。
- 密钥：API Key 存 macOS 钥匙串，普通设置和歌曲元数据存 SQLite。
- 本地工具：不内置 `yt-dlp` 和 `ffmpeg`，启动时检测并提示。
- 音频：默认保存到 `~/Music/Needle`，默认格式 `m4a`。
- 歌曲库：支持播放、删除、在 Finder 中显示文件。

## 验收标准

- 前端能构建并展示简洁三栏布局和底部播放器。
- 工具检测能区分 `yt-dlp` 和 `ffmpeg` 是否可用。
- AI 设置能保存服务商、Base URL、模型，并将 API Key 写入钥匙串。
- B 站搜索能返回候选卡，AI 筛选失败时有本地排序兜底。
- 试听失败时能打开原视频确认。
- 转换成功后歌曲进入“我的歌曲”，删除时同步删除记录和本地文件。
